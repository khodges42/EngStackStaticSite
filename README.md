# ENG Stack landing page

Static one-page site for the ENG Stack: Exoshell, NightShift, and Glassmind.

## Run locally

Open `index.html` in a browser, or serve the folder with any static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy

Upload the contents of this folder to GitHub Pages, Netlify, Vercel, Cloudflare Pages, S3 static hosting, or any plain static web host.

No build step. No external dependencies.
